[heart/3.vl](heart/3.vl) 
* Отображает тетраэдральную сетку. 
* Позволяет подсветить тетраэдры с указанным номером.
* Позволяет указать набор файлов графиков. 
При клике на тетраэдр или на вершину отображается график величин,
соответствующий выбранному объекту.

[heart2/scene4.vl](heart2/scene4.vl) 
* Отображает набор точек.
* Загружает значения из файла `voltage_.json`, который содержит двумерный массив скалярных значений.
Первая размерность массива - время. Проводит раскраску точек сообразно значениям и текущему времени `t`.
* Строит на точках набор треугольников, индексы которых загружаются из файла `triangles.txt`
* Отображает дополнительное множество точек типсов из файла `tipsHans.info` сообразно текущему `t`

[heart2/show_tips.vl](heart2/show_tips.vl ) 
* Отображает множество точек `типсов` из файла `tipsHans.info` сообразно текущему `t`.
